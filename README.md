# Dashboard Web Analitik Interaktif untuk menampilkan Insight dan Kesimpulan
## 🚀 Setup Environment

### 1. Instalasi Library
Sebelum menjalankan aplikasi, pastikan Anda telah menginstal seluruh dependensi (*library*) yang dibutuhkan. Buka terminal Anda dan jalankan perintah berikut:

```bash
pip install pandas matplotlib seaborn streamlit
```

### 🖥️ Run Streamlit App
Cara untuk menjalankan dashboard interaktif ini, yaitu:
Eksekusi Lokal (Terminal / Command Prompt)
Jika menjalankan proyek ini melalui komputer lokal, buka terminal dan lalu ketik perintah ini:
```
streamlit run dashboard.py
```

### Akses Dashboard
Dashboard dapat diakses di link berikut ini:
https://dashboard-cvision-ds.streamlit.app/
